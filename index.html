Sobre o projeto

Informações de filmes em um só lugar

Este projeto utiliza a API do TMDB para buscar filmes, avaliações, imagens, datas de lançamento e detalhes das produções. A interface foi criada com HTML, JavaScript e Tailwind CSS.
